# Adaptive Safety Scanner (ROS 2 / Embedded)

A velocity-aware safety monitoring system for autonomous robotics, compliant with ISO 13855 safety distance principles.

## Features (Planned)
- [ ] Real-time LIDAR perception (Slamtec C1)
- [ ] Dynamic Safety Zones based on object velocity
- [ ] Hardware Watchdog (Arduino) for Fail-Safe operation
- [ ] Visualization in RViz2

## Tech Stack
- ROS 2 Humble
- Python (scikit-learn for clustering)
- Arduino (C++)
